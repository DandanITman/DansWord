# DansWord Feature Log

Word-processor only. Priorities: **V1** (shipped) | **V1+** (in progress) | **Future**

## V1.0 (Shipped)

- [x] Windows desktop app (Electron)
- [x] Home screen: recent files, pinned, templates
- [x] Ribbon: File, Edit, Insert, Design, Page Layout, Review, View
- [x] Backstage File menu
- [x] Rich text editing (TipTap)
- [x] Bold, italic, underline, strike, highlight
- [x] Font family and size
- [x] Font color, subscript, superscript
- [x] Paragraph indent, line spacing, borders, and shading
- [x] Alignment, bullet/numbered lists, headings
- [x] Tables, images, hyperlinks
- [x] Native `.dansword` format
- [x] DOCX import/export
- [x] TXT open/save
- [x] Auto-save
- [x] Find (Ctrl+F)
- [x] Word count, zoom, status bar
- [x] Light/dark theme
- [x] Print + PDF export (via print dialog)
- [x] Spell check (browser native)
- [x] Custom app icon

## V1+ (Shipped)

- [x] Page setup — size, orientation, margin presets
- [x] Headers and footers + page number display (measured pages)
- [x] Navigation pane
- [x] Find & Replace (Ctrl+F / Ctrl+H)
- [x] Format Painter
- [x] Quick/custom Styles gallery + Style Editor
- [x] Insert Table of Contents
- [x] Comments sidebar
- [x] Track Changes + Accept All / Reject All
- [x] Local revision history
- [x] Ruler
- [x] Focus mode
- [x] Document envelope v2
- [x] **Page breaks** (Insert → Page Break)
- [x] **Watermarks** (Layout → Watermark)
- [x] **RTF import/export**
- [x] **Document properties** (title, author, subject, keywords, company)
- [x] **Multi-page guides** — visual page boundary lines + page count
- [x] Insert date
- [x] **Auto-updating Table of Contents** — live TOC block refreshes as headings change
- [x] **Inline anchored comments** — selection highlight + sidebar Go to anchor
- [x] **Accept/reject individual track changes** — Review ribbon Accept / Reject
- [x] **HTML export** — File → Export as HTML
- [x] **Hunspell multi-language spell check** — English, German, Spanish, French
- [x] **Hunspell spelling suggestions** — right-click misspelled words
- [x] **DOC legacy format (`.doc`)** — LibreOffice conversion with text fallback
- [x] **Dedicated PDF export** — Electron PDF engine
- [x] **Mail merge** — CSV data to batch DOCX output
- [x] **LAN collaboration** — local WebSocket document sync
