Why DansWord exists

🎓 Non-profit & educational

DansWord is a community education project — not a commercial product. The goal is to help people learn writing, formatting, and software literacy without a paywall.

🌍 Free Word alternative

Familiar ribbon-style editing, spell check, templates, and DOCX support — without Microsoft 365 subscriptions or license keys.

🔧 Open source for everyone

The full source code is public. Students, contributors, and curious developers can read it, learn from it, fix bugs, and add features.

What you get today

  • Ribbon UI like Word — File, Edit, Insert, Design, Page Layout, Review, View
  • Rich text: bold, italic, lists, tables, images, links, styles
  • Open & save DOCX, RTF, HTML, TXT, and native .dansword files
  • Spell check, find & replace, word count, zoom, light/dark theme
  • Templates for letters, reports, and resumes
  • Runs locally on Windows — your documents stay on your machine

Open source — anyone can edit

DansWord is licensed for open collaboration. Clone the repo, run it locally, send pull requests, or fork it for your own classroom or club project.

git clone https://github.com/DandanITman/DansWord.git cd DansWord && npm install && npm run dev

See CONTRIBUTING.md for how to contribute. Tests run automatically on every pull request.
